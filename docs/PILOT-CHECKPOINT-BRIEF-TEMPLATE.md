# Pilot Checkpoint-Brief Template (v1)

*Checkpoint Date: YYYY-MM-DD / Version: 1.0*
*Prepared By: CMO / Account Lead*
*Audience: Executive Sponsor, Pilot Owner, Delivery Leads*

## 1. Purpose
Use this template for a short checkpoint brief before or after governance checkpoints.
It should align stakeholders on current status, critical risks, required decisions, and near-term actions.
