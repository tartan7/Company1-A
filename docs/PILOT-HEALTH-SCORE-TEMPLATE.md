## 3. Health-Score Framework

**Scoring scale (per dimension):**
- `5 = Strong` (on track, low intervention needed)
- `4 = Stable` (minor issues, contained)
- `3 = Watch` (material risk emerging)
- `2 = At Risk` (active performance or delivery risk)
- `1 = Critical` (off-track, immediate escalation required)

**Weighting model (default):**

| Dimension | Weight (%) | Score (1-5) | Weighted Score |
| :--- | ---: | ---: | ---: |
| Outcome Performance (KPI attainment) | 35 | [ ] | [Weight x Score] |
| Delivery Execution (milestones/scope) | 25 | [ ] | [Weight x Score] |
| Operational Reliability (process/tooling) | 15 | [ ] | [Weight x Score] |
| Risk and Dependency Posture | 15 | [ ] | [Weight x Score] |
| Stakeholder Confidence and Adoption | 10 | [ ] | [Weight x Score] |
| **Total** | **100** |  | **[Total / 5 = %]** |

**Overall Health Score (%):** [0-100]

**Overall Health Band:** Green / Yellow / Red

Band guidance:
- `Green: 80-100` -> on track, standard monitoring
- `Yellow: 60-79` -> recovery plan required, leadership visibility
- `Red: <60` -> escalation path active, corrective plan mandatory
